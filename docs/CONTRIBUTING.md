# Contribution guidelines for consistency and code quality.

## Branch Naming

Use the following prefixes for branches:

- `feature/` - New features
- `fix/` - Bug fixes

## Commit Messages

Follow the Conventional Commits guidelines:

- `feat:` for new features
- `fix:` for bug fixes

## Pull Requests

- Ensure tests are updated and passing.
- Add documentation if adding new features.
